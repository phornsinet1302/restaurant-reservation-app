# restaurant-reservation-app
📌 Project Title
Restaurant Order & Reservation App
📌 Description
Smart Restaurant Reservation &amp; Ordering System — A mobile-first application that allows customers to discover restaurants, view digital menus, reserve tables, pre-order food, and make online payments, while restaurants manage orders, reservations, and promotions through a dashboard.
